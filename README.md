# English vocabulary exercises

## Basic requirements:

 - grunt
 - npm
 - bower

## Installation
 
```$: git clone https://github.com/CombatCode/test.git```

```$: cd test/```

```$: npm install```

```$: bower install```


## Launch

### By Grunt web server
```$: grunt server```


### Manually in the browser
```$: grunt build```

```$: cd dist/```

Open index.html in your favorite browser
